DBMS Terminologies

  • Data
  • Information
  • Metadata
  • Data Directory
  • Database
  • Table
  • Tuple(s)
  • Field(s)
  • Domain(s)
  • Constraint(s)
  • Key(s)
Data: Raw facts and figures.
Information: Processed data that has meaning.
Metadata: Data about data, providing information about other data.
Data Directory: A directory that stores metadata and information about the structure of databases.
Database: A structured collection of data.
Table: A structured representation of data in rows and columns.
Tuple(s): A row in a table that contains a set of related data.
Field(s): A column in a table that represents a particular attribute or property.
Domain(s): The set of possible values for a given attribute.
Constraint(s): Rules or conditions applied to data to enforce integrity.
Key(s): A field or combination of fields that uniquely identify a record in a table.
